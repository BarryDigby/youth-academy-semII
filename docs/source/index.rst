Genomics Data Science
=====================

This website contains all of the necessary materials to run NUIG Youth Academy Genomics Data Science course. 

The materials for each week will be released as the workshop progresses.

The website can be accessed using the shortened URL below. Please type this into your broswer

.. code-block:: bash

   https://tinyurl.com/YAGDS2022

Week 1
------

1. Introduction to DNA

2. Visualising genomes in IGV

3. Central Dogma of biology

4. Investigating a sequence of DNA (Worksheet)

5. RStudio cloud set up


Week 2
------

1. Recap of RStudio Basics

2. Introduction to variables, vectors and lists

3. Introduction to dataframes

4. Working with dataframes

5. Stroke dataset worksheet

Week 3
------

1. Introduction to RNA-Seq

2. High Throughput Sequencing

3. Viewing BAM files in IGV

4. Interpreting Histogram, Boxplot Gene Expression Data
